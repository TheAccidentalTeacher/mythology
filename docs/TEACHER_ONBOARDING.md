## üîç CONTENT MODERATION

### **Automatic Moderation**

All student content is automatically scanned for:
- Inappropriate language
- Violent content (beyond age-appropriate mythology)
- Hate speech
- Sexual content
- Bullying

**How it works:**
1. Student submits content (character, story, etc.)
2. AI moderation checks content (OpenAI Moderation API)
3. If flagged:
   - Content is **NOT saved**
   - Student sees: *"Content violates guidelines"*
   - You see flagged attempt in **"Flagged Content"** tab

---

### **Manual Review Queue**

**When to review:**
- Student appeals auto-moderation
- Borderline content needs human judgment

**How to review:**
1. Dashboard ‚Üí **"Flagged Content"**
2. Read content + flagging reason
3. Decision:
   - **Approve:** Content goes live
   - **Reject:** Student must revise
   - **Contact Parent:** For serious violations

---

### **Content Policy Quick Reference**

‚úÖ **ALLOWED:**
- Mythology violence (battles, monsters defeating heroes)
- Character deaths (age-appropriate context)
- Mythology-accurate descriptions (e.g., "Medusa turns people to stone")
- Fantasy weapons (swords, lightning bolts)

‚ùå **NOT ALLOWED:**
- Graphic gore/torture
- Sexual content
- Real-world violence
- Hate speech
- Bullying/harassment

*See [CONTENT_POLICY.md](./CONTENT_POLICY.md) for full guidelines.*

## üé® AI IMAGE GENERATION

Students can generate AI images for characters/creatures.

### **How It Works:**

1. Student creates character
2. Clicks **"Generate Image"**
3. AI creates image based on description
4. Student can:
   - Accept image
   - Regenerate (different style)
   - Upload own image instead

### **Teacher Controls:**

**Enable/Disable AI:**
- Dashboard ‚Üí **"Classroom Settings"**
- Toggle: **"Allow AI Image Generation"**

**Rate Limits:**
- 10 generations per day per student
- Prevents API cost overruns

**Monitor Usage:**
- Dashboard ‚Üí **"AI Usage"** tab
- See which students use AI most

**Cost:**
- ~$0.04 per image (DALL-E 3)
- Budget: ~$16 for 40 students (10 images each)

## üÜò TROUBLESHOOTING

### **Common Issues:**

#### **"Student can't log in"**
- Reset password: Dashboard ‚Üí Students ‚Üí **"Reset Password"**
- Check email spelling
- Verify join code was used during signup

#### **"Student can't see their mythology"**
- Impersonate student to see their view
- Check visibility settings
- Verify not accidentally deleted

#### **"Content keeps getting flagged"**
- Review content in **"Flagged Content"** tab
- Check for inappropriate language
- If false positive, manually approve

#### **"AI generation not working"**
- Check classroom settings: AI enabled?
- Check student daily limit (10 max)
- Verify OpenAI API key configured (admin)
